This code scrapes the Codem page and returns the data of a CUIL

### Config in .env

```
PORT=''
URL_CODEM=''
DOWNLOAD_PDF_PATH=''

```

### Dependencies utilized

```
Express.js | TypeScript | Puppeteer | pdf-parse | Dotenv
```

### Init project

```
npm run dev:start
```
